---

플레이스홀더 설명:
- `{번호}`: 문제 번호 (1, 2, 3...)
- `{난이도}`: 난이도 이모지와 텍스트 (예: `🟢 쉬움` / `🟡 보통` / `🔴 어려움`)
- `{빈칸문장}`: 핵심 단어를 `___`로 대체한 문장 또는 코드 블록. 빈칸이 여러 개일 수 있음.
- `{이미지}`: 이미지가 있을 경우 `![alt](originalSrc)` 형태로 삽입, 없으면 생략
- `{출처}`: relativePath
- `{정답목록}`: 각 빈칸의 정답을 ①②③... 형태로 나열 (빈칸 수에 맞게 작성)
- `{해설}`: 왜 그 답인지 간단한 설명

---

### 문제지 형식

```
## Q{번호}. {난이도}

빈칸을 채우시오.

{빈칸문장}

{이미지}

> **출처:** `{출처}`

**📝 내 답:** (빈칸 수에 맞게 작성하세요)

---
```

```
**정답:**

{정답목록}

**해설:**

{해설}

> **출처:** `{출처}`

---
```
