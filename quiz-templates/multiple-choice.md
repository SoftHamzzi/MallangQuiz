---

플레이스홀더 설명:
- `{번호}`: 문제 번호 (1, 2, 3...)
- `{난이도}`: 난이도 이모지와 텍스트 (예: `🟢 쉬움` / `🟡 보통` / `🔴 어려움`)
- `{문제내용}`: 질문 본문 (예: "다음 중 올바른 것은?")
- `{이미지}`: 이미지가 있을 경우 `![alt](originalSrc)` 형태로 삽입, 없으면 생략
- `{보기A}` ~ `{보기D}`: 각 보기 내용
- `{출처}`: relativePath
- `{정답번호}`: 정답 보기 번호 (A/B/C/D)
- `{해설}`: 왜 정답인지, 오답은 왜 틀렸는지 설명

---

### 문제지 형식

```
## Q{번호}. {난이도}

{문제내용}

{이미지}

- A. {보기A}
- B. {보기B}
- C. {보기C}
- D. {보기D}

> **출처:** `{출처}`

**📝 내 선택:** (  )

---
```

### 해답지 형식

```
## Q{번호}. {난이도}

**문제:** {문제내용}

{이미지}

- A. {보기A}
- B. {보기B}
- C. {보기C}
- D. {보기D}

**정답: {정답번호}**

**해설:**

{해설}

> **출처:** `{출처}`

---
```
