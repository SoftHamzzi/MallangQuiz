플레이스홀더 설명:
- `{번호}`: 문제 번호 (1, 2, 3...)
- `{난이도}`: 난이도 이모지와 텍스트 (예: `🟢 쉬움` / `🟡 보통` / `🔴 어려움`)
- `{문제내용}`: 질문 본문
- `{이미지}`: 이미지가 있을 경우 `![alt](originalSrc)` 형태로 삽입, 없으면 생략
- `{출처}`: relativePath
- `{정답}`: 상세한 답변과 설명

---

### 문제지 형식

```
## Q{번호}. {난이도}

{문제내용}

{이미지}

> **출처:** `{출처}`

**📝 내 풀이:**

<풀이를 여기에 작성하세요>

---
```
