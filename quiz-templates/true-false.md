---

플레이스홀더 설명:
- `{번호}`: 문제 번호 (1, 2, 3...)
- `{난이도}`: 난이도 이모지와 텍스트 (예: `🟢 쉬움` / `🟡 보통` / `🔴 어려움`)
- `{명제}`: 참 또는 거짓으로 판단할 문장
- `{이미지}`: 이미지가 있을 경우 `![alt](originalSrc)` 형태로 삽입, 없으면 생략
- `{출처}`: relativePath
- `{정답}`: O 또는 X
- `{해설}`: 왜 O 또는 X인지 설명

---

### 문제지 형식

```
## Q{번호}. {난이도}

다음 명제가 참(O)인지 거짓(X)인지 판단하라.

> {명제}

{이미지}

> **출처:** `{출처}`

**📝 내 답:** O / X

---
```

### 해답지 형식

```
## Q{번호}. {난이도}

**문제:** 다음 명제가 참(O)인지 거짓(X)인지 판단하라.

> {명제}

{이미지}

**정답: {정답}**

**해설:**

{해설}

> **출처:** `{출처}`

---
```
